## 🚀 How to Run

Clone the repository

```bash
git clone https://github.com/CodeWithIshwar-dev/python-practice.git
```

Move into the project

```bash
cd python-practice
```

Run any Python file

```bash
python filename.py
```

or

```bash
python3 filename.py
```
